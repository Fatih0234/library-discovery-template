# Starter Prompt

Use this prompt when starting a new library exploration from this template.

```text
Read AGENTS.md, PHILOSOPHY.md, SPEC.md, TEMPLATE.md, and TASKS.md.
Research <library> from official docs and first-party examples.
Create examples/<library>/docs.md as a curated beginner-focused notes file.
Then generate examples/<library>/output.md using TEMPLATE.md.
Keep it minimal, concept-first, and focused on one first learning project.
```

If the library is Python-related, prefer `uv` for environment and package management.
